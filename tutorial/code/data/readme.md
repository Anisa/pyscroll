Quest Demo
==========

This is a demonstration of using pyscroll and pytmx together to create a game.

Controls:
- Arrow Keys: movement
- Plus/Minus (+/-): zoom the camera

The window is resizeable, camera can zoom in and out using +/- keys.

Enjoy!


Tileset
=======
